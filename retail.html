Lego Wave - NYC Flagship Store

A project for Pixel Artworks that enable customers to interact with Lady Liberty on screens above the tills. When you wave long enough she waves back at you and plays one of many fun animations.
This installation uses Azure Kinect Developer Kit to detect player interactions. The alembic animations are all rendered in real-time.

Facebook table

An interactive touch-table installation for Inition.
In this project we used a MultiTaction touch table and Unity to create an interactive data visualisation app for Facebook.
Acrylic pucks with markers beneath them were used to trigger different info panels for sports.
The table was exhibited at an event in France and then in the offices of Facebook.

Video wall - Westfield London

An interactive video wall installation for Inition.
The project consisted of using Kinect v2 to create 3 interactive experiences.
The three 5 x 2.5 meter screens were part of multiple installations and the event took place in both Westfield London (27 – 29 March 2015) and Westfield Stratford City (2 - 4 March 2015).

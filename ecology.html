Drop In the Ocean - Tribeca Film Festival 2019

Networking backend development, VR setup process and integration of Captury - a markerless motion captury system for Target3D and Vision3.
4 users can go through the experience at the same time and they can see and interact with each other and the world around them. The experience takes place inside a large jellyfish installation.
There are 3 different deployment targets:
- Controller application: A UWP sideloaded app installed on all headsets
- Server: A Linux app that runs on the Captury server computer
- Controller: A Windows app that runs on a tablet that’s used to control the experience

Rhino Conservation - Cisco Connected

This is an Augmented Reality project commissioned by Cisco. It was displayed in an event in Las Vegas as part of their Connected Conservation campaign.
The rhinos are placed procedurally, so every time the user experience is different. The goal is to raise awareness towards the ongoing extinction of these majestic creatures.

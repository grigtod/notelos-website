VR Payments - WorldPay

This VR project for Worldpay aims to demonstrate payments for products in VR. There are 3 different objects to purchase and each one has a different cost level. The more expensive the item, the more effort it requires to input the pin code, so the possibility of accidental purchases is reduced.
